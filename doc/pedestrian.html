Parameters

Parameter name Use for Values
Pedestrian number Set the number of pedestrian generate for the simulation Min : 1. Half will try to reach the left side of the simulation space, the other half the right side
Space length The length of the horizontal side Min 5
Space width The length of the vertical side Min 5
Bottleneck size Size of the hole between the two part of the space Min : 0. With 0 there is no way to switch side for any agent. Beside, reliant on the Interaction strength and the Range of the repulsive interactions, a non null but to small bottleneck cannot be pass. If you want no bottleneck, you need to choose a size superior or equal to the Space width
